This project fulfills the requirements for Image Processing at the University of Utah project #5. Any documentation beyond what is located in the code can be found in the document write up attached with the ZIP file. 

This project is implemented in matlab. To run the driver you must add the entire project repository to your matlab run the Main.m. You can change in the input parameter files in the main function. All input parameters used in the report are detailed in the caption of the image. You must change the directory of the add path function to whatever directory you have placed the “project5” directory into. 

All other details of the implementation are in the PDF document as well as all output images.
